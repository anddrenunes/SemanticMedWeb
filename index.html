The Semantic MedWeb
is a framework that provides semantic integration of Translational Research databases.
The Semantic MedWeb
iis based on the Multilevel Healthcare Information Modeling (MLHIM) specifications.
Semantic Web technologies
are the solution to the integration of Translational Research databases. The Semantic Med Web adds the innovation of the multilevel modeling software architecture to it.
Translational Research needs database integration because... 
This is the key to promote the interface bewteen basic and clinical science
Innovation is important...
To overcome current challenges in semantic integration of distributed databases.

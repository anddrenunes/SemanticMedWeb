SemanticMedWeb API 1.0

This is the official documentation for the SemanticMedWeb API version 1.0

Implementations on any platform must adhere to this API. The reference implementation is on an eXist-db 2.0 platform.

Authentication/Authorization:

OAuth2 is a protocol that lets external apps request authorization to private details without getting their password. This is preferred over Basic Authentication because tokens can be limited to specific types of data, and can be revoked by users at any time. No unauthenticated requests will be honored.

SemanticMedWeb

This is the entry page into Semantic Medical Web Application (SemanticMedWeb).
SemanticMedWeb is the only healthcare application designed specifically for semantic interoperability for healthcare. It is based on the innovative multi-level modelling prinicples developed over the past three decades of research and development in semantic interoperability in healthcare. Based on leading edge XML technologies, SemanticMedWeb provides complete flexibility and adaptability. Semantic MedWeb is built on the Ismene platform, the most customizable healthcare application available; anywhere.
You can adapt SemanticMedWeb to any type of application including practice specific EMR, mobile device data collection, data aggregation application, etc. Maintaining the ability to have complete semantic interoperability across all applications. SemanticMedWeb ships with all source code included.
